


Git -> Sistema de controle de versão distribuido
SCM -> Source Control Management
Criado por linus torvalds (2005), para auxiliar no dDesenvolvimento do Linux.

- Gerência de Configuração de Software
	- Durante um projeto de desenvolvimento
		* O que mudou e quando ?
		* Por que Mudou ?
		* Quem Mudou ?
		* Podemos reproduzir a mudança ?
		
	* Identificação
	* Documentação
	* Controle
	* Auditoria

	
	-Artefatos:
		- Código fonte
		- Documentação do software
		- Manual do usuário
	
- Controle de Versão

	* Merge
	* Commit
	*

- Git
- Subversion (SVN)
- Mercurial (HG)
- CVS - Concurrent Versioning System
- Bazaar
