$ pwd -> mostra o diretorio atual $ ls ->Lista todos os arquivos do diretório $ ls -la ->Lista todos os arquivos do diretório inclusive os ocultos $ clear -> Limpa a tela $ mkdir -> cria um diretorio $ cd $ Tree .git/ -> Mostra a arvore mostra a estruturade pastas $ echo "# git_command" >> README.md $ git init -> Cria um repositorio vazio dentro da pasta que estiver posicionado. $ git status -> Mostra as diferenças entre diretorio do projeto (Diretorio de trabalho) e a area de staging (conteiner do git). $ git add -> inclui alterações dos arquivos da area de trabalho para a area de stanging. $ git add -A -> inclui todos arquivos alterações dos arquivos da area de trabalho para a area de stanging. $ git config --global user.email "datacrusher@outlook.com" $ git config --global user.name "datacrusher" $ git commit - m 'Primeiro Commit' $ git log Para subir para o github: $ git remote add origin https://github.com/Datacrusher/git_command.git $ git push -u origin master
