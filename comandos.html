#Branch é uma nova linha de desenvolvimento (linha de tempo)independente da principal (master). [traduções: Seção, ramo, galho , subdividir-se ] #visualizando as linhas do temo criadas.apenas digite o comando $git branch #Criando uma nova linha de tempo (branch) $git branch # Ex: $git branch line1 # O comando Cat serve para criar ou mostrar o conteúdo de um arquivo. #no exemplo abaixo o conteudo mostra o ponteiro para o ultimo commit $cat .git/refs/heads/master $cat .git/refs/heads/line1 #HEAD (area de trabalho staging) #mosca o local atual de trabalho (staging) $cat .git/head # mudar de Branch (linha de tempo) # o comando abaixo Move o Head do branch atual (master) para o (line1) criado anteriormente. $git checkout line1 #verificando o head atual $cat .git/head #Ignorar Arquivos #Os nomes de arquivos/diretórios ou extensões de arquivos listados no arquivo .gitignore não serão adicionados em um repositório.
